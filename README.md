# Proyecto de Ajedrez

Este proyecto es un simulador de ajedrez en Java que permite visualizar un tablero de ajedrez y las piezas en movimiento a partir de un formato PGN (Portable Game Notation). Se utiliza una interfaz gráfica construida con Swing para representar el estado del juego.

## Requisitos

- Java Development Kit (JDK) 8 o superior.
- Un entorno de desarrollo integrado (IDE) como Eclipse, IntelliJ IDEA o NetBeans.

## Estructura del Proyecto

El proyecto está organizado en paquetes, cada uno de los cuales cumple una función específica:

- *Modelo*: Contiene las clases que representan la lógica del juego, incluyendo piezas (como Torre, Rey, etc.), el tablero y el lector de archivos PGN.
- *Controlador*: Contiene la clase JuegoControlador, que gestiona la lógica del juego y la interacción entre el modelo y la vista.
- *Vista*: Contiene las clases responsables de la interfaz gráfica, como VisualTablero, que representa el tablero de ajedrez en la ventana.

## Cómo Ejecutar el Proyecto

Para ejecutar el proyecto, sigue estos pasos:

1. Clona o descarga el repositorio del proyecto.
2. Abre el proyecto en tu IDE preferido.
3. Asegúrate de que todas las dependencias estén configuradas correctamente.
5. Ejecuta el método MenuPrinciapl de esta clase contenida en el paquete vista.

El código dentro de TestVisualPiezas crea un tablero de ajedrez y lo muestra en una ventana utilizando un contenido de ejemplo en formato PGN que está en un archivo
